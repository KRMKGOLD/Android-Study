# Android-Study

## Introduce

- 깃허브를 처음 시작하고 잡다하게 분리된 안드로이드 Repo 모음
- Android에 대해 공부한 여러가지
- Java, Kotlin 뿐만 아니라 관련된 내용들도 포함된 Repo들이 모여있음

## Necessary Study

- MVP, MVVM, Clean Architecture
- Android Architecture Components, Jetpack
- Socket & Multi Chat
- RxJava, RxKotlin, RxAndroid ...
- Coroutine
